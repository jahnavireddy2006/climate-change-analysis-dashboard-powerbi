# 🌍 Climate Change Analysis Dashboard

## 📌 Overview

The Climate Change Analysis Dashboard is an interactive Power BI project designed to analyze key environmental indicators such as CO₂ emissions, rainfall patterns, temperature changes, and regional contributions. The dashboard helps users understand climate trends through interactive visualizations.

## 🎯 Objectives

- Analyze country-wise CO₂ emissions.
- Compare regional CO₂ contributions.
- Study rainfall trends over multiple years.
- Analyze year-wise temperature changes.
- Present climate data using interactive dashboards.

## 📊 Dashboard Features

- 🌍 Country-wise CO₂ Emission Analysis
- 🥧 Regional CO₂ Contribution
- 🌧️ Rainfall Trend Analysis
- 🌡️ Year-wise Temperature Analysis
- Interactive Power BI Visualizations

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel / CSV Dataset
- Data Visualization
- Business Intelligence
- Data Analysis
